Getting Started with React Hooks

React Hooks revolutionized how we write components. They let you use state and other React features without writing a class. Let's dive into the two most essential hooks: `useState` and `useEffect`.

The `useState` Hook

The `useState` hook is your tool for adding state to functional components. In the past, you needed a class component to manage state. Now, it's as simple as calling a function.

import React, { useState } from 'react';

function Counter() {
  // Declare a new state variable, which we'll call "count"
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>You clicked {count} times</p>
      <button onClick={() => setCount(count + 1)}>
        Click me
      </button>
    </div>
  );
}

Building a REST API with Node.js and Express

Node.js and Express are a powerful combination for building fast and scalable APIs. A REST API is an architectural style that defines a set of constraints for creating web services. Let's build a simple "Hello World" server.

Setting Up The Server

First, make sure you have Node.js installed. Then, initialize a new project and install Express.

npm init -y
npm install express

Now, create a file named `server.js` and add the following code. This sets up a basic server that listens on port 3000 and responds to requests to the root URL.

const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello World!');
});

app.listen(port, () => {
  console.log(`Example app listening at http://localhost:${port}`);
});

The Basics of Smart Contracts with Solidity

Solidity is the most popular language for writing smart contracts on Ethereum and other compatible blockchains. A smart contract is like a regular contract, but instead of being written on paper, it's written in code and runs on the blockchain.

Your First Smart Contract

Here is a simple "HelloWorld" smart contract. It stores a piece of text and has a function to update it. Note the `pragma` line, which specifies the compiler version, and the use of state variables and functions.

// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract HelloWorld {
    string public greet = "Hello World!";

    function setGreeting(string memory _newGreet) public {
        greet = _newGreet;
    }
}
